## Notes
- Files:
```Javascript
server/ // Back end
    index.js // Back end
    db.js // Back end

public/ // Back end
    index.html // Back end
    style.css // Back end
    bundle.js // Front end

client/ // Front end
    Footer.js // Front end
    Header.js // Front end
    React // Front end
```
- Backend
    - Node - allows execution of JavaScript
        - Express - framework for writing web servers
        - `http://`
        - Listening on port 8080

    - Postgres - object-relational database system
        - `postgres://`
        - Listening on port 5432

- Frontend
    - React - to build user interfaces
        - Babel will convert JSX into JS for us
        - Webpack will put .js files and "bundle" it for us
